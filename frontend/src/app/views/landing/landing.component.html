VIRA Project Manager

A open source software with multiple tools to manage all your projects. Your can create kanban boards, chat with users, interact with a calendar, save all your documentation.

Kanban

A visualization tool that enables you to optimize the flow of your work.

Calendar

A calendar to manage all your events like calls, meetings, etc.

Chat

A place where you can talk with other users registered in the plattaform.

Docs

A site where all documentation is saved.
